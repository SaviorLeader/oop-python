# Sadə Python Döyüş Oyunu

Bu layihə Python dilində OOP (obyekt yönlü proqramlaşdırma) öyrənmək üçün hazırlanmış sadə döyüş simulyatorudur.  
İki personaj (`Fara` və `Bro`) bir-birinə hücum edir və sağlamlıqlarına görə zərər alırlar.

---

## Texnologiyalar
- Python 3.x
- OOP (Classes və Objects)

---

## Layihənin Təsviri
- `Fara` və `Bro` adlı iki personaj mövcuddur.  
- Hər personajın:
  - `name` (adı)
  - `health` (sağlamlığı)
  - `attack_power` (hücum gücü) var.  
- Personajlar bir-birinə `attack()` funksiyası ilə hücum edə bilirlər.
