# Sadə Python Döyüş Oyunu

Bu layihə Python dilində OOP (obyekt yönlü proqramlaşdırma) öyrənmək üçün hazırlanmış sadə döyüş simulyatorudur.  
İki personaj (`Fara` və `Bro`) bir-birinə hücum edir və sağlamlıqlarına görə zərər alırlar.

---

## Texnologiyalar
- Python 3.x
- OOP (Classes və Objects)

---

## Layihənin Təsviri
- `Fara` və `Bro` adlı iki personaj mövcuddur.  
- Hər personajın:
  - `name` (adı)
  - `health` (sağlamlığı)
  - `attack_power` (hücum gücü) var.  
- Personajlar bir-birinə `attack()` funksiyası ilə hücum edə bilirlər.


## Necə işə salmaq olar

1. Python 3 quraşdırılmış olmalıdır
2. Faylı işə salın:

```bash oop.py


##  Python necə yükləmək olar?

1. Brauzerdə Google açın
2. **Python** sözünü axtarın
3. Rəsmi sayta daxil olun: https://www.python.org
4. **Download Python** düyməsinə basın
5. Yükləmə zamanı **"Add Python to PATH"** seçimini işarələyin
6. Quraşdırmanı tamamlayın
